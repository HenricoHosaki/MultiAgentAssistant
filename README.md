# MultiAgentAssistant

Multi-agent Customer Support (SAC) assistant — answers questions about
**Products, Deliveries, and Payments** using specialized agents + RAG.

> Architecture, decisions, and roadmap: see [docs/PLANEJAMENTO.md](./docs/PLANEJAMENTO.md) and [docs/ROADMAP_ESTUDOS.md](./docs/ROADMAP_ESTUDOS.md).

## Tech stack

- **Python 3.12** · **uv** (dependency management)
- **CrewAI** (multi-agent orchestration)
- **Gemini** (LLM + embeddings, via LiteLLM)
- **Chroma** (local vector store for RAG)
- _FastAPI and React — added in upcoming phases_

## Prerequisites

- Gemini API key ([Google AI Studio](https://aistudio.google.com/apikey)) — the free tier is enough

### 1. Install `uv` (dependency manager)

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
Close and reopen the terminal, then confirm: `uv --version`

### 2. Install Python (via uv)

```powershell
uv python install 3.12
```

### 3. Install the CrewAI CLI

```powershell
uv tool install crewai
uv tool update-shell
```
Confirm: `crewai version`

## Setup

The crew code lives in `sac_assistant/` (generated by the CrewAI CLI).

1. Clone the repository
2. Enter the crew folder: `cd sac_assistant`
3. Create the `.env` file with:
   ```
   MODEL=gemini/gemini-2.5-flash
   GEMINI_API_KEY=your_key_here
   ```
4. Install dependencies: `crewai install`

## How to run

```powershell
cd sac_assistant
crewai run
```

Currently runs the `products_specialist` agent, answering a single hardcoded
question (set in `main.py`) grounded in the Products knowledge base, with source citations.

## Project structure

```
MultiAgentAssistant/
├── docs/                       # PLANEJAMENTO.md, ROADMAP_ESTUDOS.md
├── README.md
└── sac_assistant/              # CrewAI project
    ├── .env                    # secrets (not committed to git)
    ├── pyproject.toml          # dependencies + commands
    ├── knowledge/              # RAG knowledge base (Phase 1: products/)
    └── src/sac_assistant/
        ├── main.py             # entry point (run/train/test)
        ├── crew.py             # assembles the Crew (Agents + Tasks + Process)
        ├── config/
        │   ├── agents.yaml     # defines the agents
        │   └── tasks.yaml      # defines the tasks
        ├── rag/
        │   ├── ingest.py       # chunking + embeddings -> Chroma
        │   └── retriever.py    # similarity search against Chroma
        └── tools/
            └── knowledge_search_tool.py  # CrewAI Tool wrapping the retriever
```

## Environment variables

| Variable | Description |

| `GEMINI_API_KEY` | Gemini API key — **never commit** (stays only in `.env`) |
| `MODEL` | Model used (e.g., `gemini/gemini-2.5-flash`) |
